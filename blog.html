Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Think of it as the whole package! It gives you everything inside an element,
including HTML tags, visible text, hidden text, and even CSS styles that are directly within the element.

innerText: This one's more selective. It gives you only the visible text content of an element,
excluding HTML tags, hidden text (text hidden with CSS), and CSS styles. What you see is what you get!

textContent: This is the most inclusive text getter. It gives you all the text content within an element,
including text that's hidden with CSS, but excluding HTML tags and CSS styles.
