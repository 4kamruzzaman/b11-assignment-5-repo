Question-1: What are the different ways to select an element in the DOM?

Like, if you know the ID, you do document.getElementById('theId'). Easy peasy.

Or, if you wanna grab something by its class, you can use document.querySelector('.theClass'). That gets the FIRST one.

If you want ALL of them with that class, it's document.querySelectorAll('.theClass'). That gives you like, a list of them.

And there's also document.getElementsByTagName('p') to get all the <p> tags. You get the idea.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML = Gives all things that under a element. Like tags, text, CSS etc.
innerText = It gives only the text that are visible except the hidden text, Tags, CSS etc.
textContent = It gives all the text that are under a element including hidden text, Tags, CSS etc.

Question-5: How do you create, add, and remove elements using JavaScript?

Create: To create a new element, document.createElement('div') can be use. 'div' can be replaced with any tag name.

Add: This helps to adding attribute to an element. E.g. parentElement.appendChild(theNewThing)

Remove: This can be used for remove/delete an element. E.g. element.remove(). It will be removed from the DOM.
